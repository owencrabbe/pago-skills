# PAGO Skills

Free, downloadable AI skills for everyday life and real estate. Drop them into Claude or any AI agent and put them to work in minutes.

Built and maintained by [PAGO](#about-pago).

## What is a skill

A skill is a small instruction file that teaches an AI agent how to do one job well. Instead of writing the same long prompt every time, you load the skill once and the agent knows the process, the questions to ask, and the format to deliver.

Every skill in this repo is a single `SKILL.md` file. No code to install. No account required. Free to use and free to share.

## Skills in this repo

| Skill | What it does | Best for |
| --- | --- | --- |
| [lease-review](skills/lease-review) | Reads a lease or contract and explains it in plain English, flags risky clauses, and gives you the questions to ask before you sign | Renters, buyers, small business owners, anyone signing a document |
| [relocation-compare](skills/relocation-compare) | Compares two or more cities on cost of living, jobs, housing, taxes, and lifestyle, then gives you a clear recommendation | Anyone deciding where to move or where to open a business |
| [inbox-triage](skills/inbox-triage) | Sorts a pile of emails by urgency, summarizes each, and drafts replies you can send | Anyone drowning in email |

More skills are added regularly. Star the repo to follow along.

## How to use a skill

See [INSTALL.md](INSTALL.md) for step by step setup. The short version:

1. Open the skill folder and copy the contents of `SKILL.md`.
2. Paste it into your AI agent at the start of a chat, or save it as a skill if your agent supports skill files.
3. Tell the agent what you need. It already knows the process.

These skills work with Claude, and the instructions are written in plain language so they also work with most other capable AI assistants.
